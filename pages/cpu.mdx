# HPC

## OpenMP

_OpenMP_[^openmp] uses a fork/join model.

1. Fork[^fork]: Creates[^create] a number of parallel threads from a primary thread
   - Primary thread is always thread 0
   - No guarantee of order
2. Join[^join]: Synchronises thread termination and returns program control to
   primary
